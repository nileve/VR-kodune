h1: Oracle

Oracle (tuntud ka kui Oracle RDBMS või Oracle Database) on objekt-relatsiooniline andmebaaside haldussüsteem, mille tootja ja turustaja on Oracle Corporation.
1977. aastal asutasid sõbrad ning endised töökaaslased Larry Ellison, Bob Miner ja Ed Oates konsultatsioonifirma Software Development Laboratories (SDL). SDL töötas välja Oracle tarkvara esialgse versiooni. Nimi Oracle pärineb CIA poolt rahastatud projekti koodnimest, mille kallal Ellison varasemalt Ampexi palgalisena töötanud oli.

h2: Struktuurid

      Oracle andmebaasisüsteem hõlmab endas vähemalt ühte rakenduse isendit (instance) koos andmehoidlaga. 
      Isend sisaldab kogumit operatsioonisüsteemi protsessist ja mälustruktuuridest, mis mäluga suhtlevad. 
      Tüüpilised protsessid on PMON (process monitor), mis jälgib teisi taustaprotsesse, ja SMON (system monitor), 
      mis tegeleb isendi taastamisega ja hooldab kontrollfaile.

      Oracle andmebaaside kasutajad viitavad serveripoolsele mälustruktuurile kui SGA-le (System Global Area). 
      SGA valdab tavaliselt vahemälu informatsiooni, nagu näiteks andmepuhvreid, SQL käske ja kasutajaandmeid. 
      Lisaks mälule koosneb andmebaas võrgupõhistest taastelogidest, mis sisaldavad protseduuride ajalugu. 
      Protsessid võimaldavad taastelogisid arhiveerida, mille põhjal teostatakse andmetaastust ja andmejäljendust 
      kasutades Oracle Data Guardi. Oracle'i dokumentatsioon viitab aktiivsele andmebaasi isendile kui „jagatud mälu alale“.

h3: Versioonide nummerdus

Oracle’i tooted järgivad kohandatud numeratsiooni ja nimetamise tava. Alates Oracle RDBMS 10g avaldamisest on Oracle Corporation kasutanud „10g“ märgist kõigis peamistes tooteversioonides. Järelliited „i“, „g“ ja „c“ ei kujuta endast tegelikult madalat järku versiooninumbri osa, vaid on vastavad lühendid sõnadele „internet“, „grid“ ja „cloud“.[11] Paljud jätavad kindlatele Oracle'i tooteversioonidele viidates järelliite välja.
